# Archive

This folder contains archived files that were used during dataset preparation but are not needed for data analysis or BCO-DMO submission.

## Contents

### `old_data_backups/`
Contains backup copies of data files created during the data cleaning process:
- `BACKUP_maatea_size_cafi_taxonomy_merged_2019_2021_v2.csv`
- `BACKUP_mrb_amount_cafi_field_experiment_summer_2021_v4.csv`

These were created as safety backups and are identical to the final versions in the `data/` folder.

### `original_documents/`
Contains the original Word documents (.docx) used to create the plain text README files:
- `README_amount_project_overview.docx`
- `README_photogrammetry_metadata_v2.docx`
- `README_size_project_overview.docx`
- `README_survey_project_overview.docx`
- `README_tip_stump_comparison_dec_2019.docx`

Plain text versions (.txt) are available in the `metadata/` folder for better accessibility and version control.

## Note

These files are preserved for documentation purposes but are **not required** for data analysis or repository submission. All essential information from these files is available in the main dataset documentation.
