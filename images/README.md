# Images and Photographs

This folder contains visual documentation of the Mo'orea CAFI research project.

## Contents

### Field Work Photos
- **`field_work_1.jpg`** - Field data collection in Mo'orea
- **`field_work_2.jpg`** - Research team conducting surveys

### Representative CAFI Species Photographs

**Crustaceans:**

## Purpose

These images serve as:
1. **Visual reference** for species identification
2. **Documentation** of field methods and study sites
3. **Educational material** for presentations and publications
4. **Context** for understanding the research environment

## Notes

- All photographs taken during 2019-2021 field seasons in Mo'orea, French Polynesia
- Species photographs show live specimens collected from *Pocillopora* coral colonies
- These are **representative examples**, not a complete catalog of all species found
- For complete species lists, see CAFI taxonomy data files in the `data/` folder

## Usage

These images are provided for context and educational purposes. When using these images in publications or presentations, please cite this dataset and credit the photographers (Stier Lab, UC Santa Barbara).
